# 功能完成记录

## 2026-03

### 1. 认证模块升级（JWT + Passport + Redis 会话）

- 关键改动：
  - 登录/续签改为 access + refresh 双 token（HttpOnly Cookie）
  - Redis 会话引入 `sid` 校验，支持单会话退出与一键全下线
  - `GET /auth/status` 替代 `/auth/me`，用于登录态检查与临期自动续签
  - `POST /auth/login` 仅返回过期时间，不返回用户信息
- 影响范围：
  - `src/modules/auth/*`
  - `src/common/guards/*`
  - `src/common/redis/*`
  - `src/config/*`

### 2. 全局基础能力整理

- 关键改动：
  - 全局鉴权、全局参数校验、全局日志、全局异常处理统一注册
  - `AppModule` 装配清单拆分到 `src/setup/*`
  - `main.ts` 仅保留启动主流程
- 影响范围：
  - `src/app.module.ts`
  - `src/setup/*`
  - `src/common/*`

### 3. 日志模块落地（Winston）

- 关键改动：
  - 接入 `nest-winston`、文件滚动日志、彩色控制台日志
  - 支持四级日志：`error/warn/info/debug`
  - 新增 `AppLoggerService` 与敏感字段脱敏
- 影响范围：
  - `src/common/logger/*`
  - `src/common/interceptors/request-logging.interceptor.ts`
  - `src/common/filters/http-exception.filter.ts`

### 4. Prisma 基础集成与 RBAC 模型模块化

- 关键改动：
  - 新增 `PrismaModule/PrismaService`
  - Prisma schema 改为目录模式：`prisma/schema/schema.prisma + modules/*.prisma`
  - RBAC 按“单表单文件 + 关系文件”拆分
  - `User` 模型新增 `avatar` 字段
  - 关系动作统一为 `Restrict`（兼容 `relationMode = "prisma"`）
  - 执行迁移：`init_rbac_modular`
- 影响范围：
  - `src/common/prisma/*`
  - `prisma/schema/*`
  - `prisma/migrations/*`
  - `dosc/03-数据库处理规范.md`
  - `dosc/04-RBAC数据库设计.md`

### 6. User 模块 CRUD 落地

- 关键改动：
  - 新增 `user` 模块，提供用户增删改查接口
  - 使用软删除（`deletedAt`）
  - 新增 DTO 参数校验与唯一冲突处理
  - 模块内新增 `README.md` 文档
- 影响范围：
  - `src/modules/user/*`
  - `src/setup/app-imports.ts`
